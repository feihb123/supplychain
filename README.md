汽配供应链综合管理平台
-
![supplychain](https://github.com/feihb123/images/blob/master/supplychain.png)
1. 角色：
- 制造商
- 批发商
- 汽配商
- 平台管理
- 消费者（主要）


2. 主要技术栈
- Springboot
- thymeleaf
- websocket
- bootstrap-fileinput
- three

3.消费者功能
- 个人信息管理 （bootstrap-fileinput 上传文件存储在本地路径）
- 购物车
- 订单管理
- 预约管理
- 私信留言 （websocket 实时交互）
- 个性定制（three 3D渲染）
- 分类搜索
